#### 题目描述

一个班级里，男生比女生多 a 人，当男生和女生同时调走 b 个人后，男生数量是女生的两倍，问原来男生女生各有多少人？

#### 输入

-   第一行：单个整数 a，表示开始时男生比女生多 a 人；
-   第二行：单个整数 b，表示男生和女生同时调走 b 个人。

#### 输出

-   第一行：一个整数，表示原来有多少男生；
-   第二行：一个整数，表示原来有多少女生。

#### 数据范围

-   1≤a≤1000；
-   1≤b≤1000.

___

#### 输入样例1复制

```
1
1
```

#### 输出样例1复制

```
3
2
```

___

#### 输入样例2复制

```
7
5
```

#### 输出样例2复制

```
19
12
```
