#### 题目描述

小酷想要坐飞机去旅行，离飞机起飞还有 a 小时 b 分钟 c 秒，小酷是个争分夺秒的人，你能帮他计算还有多少秒吗？  
提示：  

-   1 分钟 = 60 秒
-   1 小时 = 3600 秒

#### 输入

输入为三行，包括三个整数 a, b, c，含义如题目所述，满足 0≤a, b, c≤100。

#### 输出

输出为四行：  

-   第一行为将 a 小时换算为秒的结果；  
    
-   第二行为将 b 分钟换算为秒的结果；  
    
-   第三行为将 c 秒换算为秒的结果；  
    
-   第四行为将 a 小时 b 分钟 c 秒换算为秒的结果。

___

#### 输入样例复制

```
1
2
3
```

#### 输出样例复制

```
3600
120
3
3723
```
