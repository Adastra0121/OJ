#### 题目描述

给定时刻 a 和时刻 b，求时刻 a 最少经过多少分钟之后才能到达时刻 b。（24 小时计时制）

#### 输入

输入为四行，包含四个整数 x1, y1, x2, y2

-   a 时刻为 x1 小时 y1 分钟  
    
-   b 时刻为 x2 小时 y2 分钟  
    

#### 输出

一个整数，表示时刻 a **最少** 经过多少分钟之后才能到达时刻 b。

___

#### 输入样例1复制

```
0
0
12
0
```

#### 输出样例1复制

```
720
```

___

#### 输入样例2复制

```
12
30
0
0
```

#### 输出样例2复制

```
690
```
