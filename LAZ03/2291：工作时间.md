#### 题目描述

小酷每天的上下班的时间不固定。某一天，他上班时间是 a 时 b 分，下班时间是当天的 c 时 d 分。请你帮小酷计算一下，它这天一共工作了多少时间？工作时间和薪水有关，你可不要算错了哦。  
本题中的计时都按 24 小时制计算。

#### 输入

输入 4 行，每行一个整数，分别表示a, b, c, d（0≤a≤c≤23，0≤b, d≤59，数据保证下班时间在上班时间之后）。

#### 输出

一行两个整数 e, f，表示工作时间是 e 小时 f 分钟。

___

#### 输入样例复制

```
3
20
7
0
```

#### 输出样例复制

```
3 40
```
